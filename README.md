# Supervised-Learning-from-Scratch-NumPy
Implementation of linear classifiers, logistic regression,  and multi-layer neural networks built from scratch using  NumPy, as part of IN4050 at the University of Oslo.

## What's covered
- Linear regression classification with MSE loss
- Logistic regression (binary + one-vs-rest multi-class)
- Softmax / multinomial logistic regression
- Multi-layer perceptron with backpropagation from scratch
- Hyperparameter tuning via grid search (learning rate, epochs)
- Final evaluation with precision, recall, and accuracy

## Tech stack
`Python` `NumPy` `Matplotlib` `Scikit-learn` `Jupyter Notebook`

## Key results
- Linear regression classifier: 0.669 precision, 0.503 recall
- Logistic regression (scaled): 0.762 validation accuracy
- Softmax classifier: 0.77 validation accuracy
- MLP with backprop: best performing model across all tasks
